Hibernate Tools - Tools for Hibernate
=====================================
version 2.1.3, 18. November 2004

Instructions
------------

Unzip to installation directory.

Hibernate-tools.jar is dependent on hibernate2.jar, available seperatly.

Other required jars are distributed in the lib subdirectory.

Note: The bin/setenv.bat has a HIBERNATE_HOME variable that needs
to point to the root of the hibernate2 directory.
