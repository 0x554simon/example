是否必须设定浏览器接受cookies?

是，阁下必须将浏览器设定至接受cookies

为何浏览器会自动记忆我的个人户口号码及密码?

这是浏览器内「 自动完成」的功能。为避免当您键入用户名称时， 电脑会自动完成您的密码输入， 您应取消浏览器的「 自动完成」 功能。

在Internet Explorer 浏览器中， 这个「 自动完成」功能会储存您过去曾经就网址、 表格和密码所输入的资料， 然后当您再一次键入这些资料时，「 自动完成」功能就会提议可能的配对资料。因此为确保您的户口资料获得适当保护， 您应取消浏览器的「 自动完成」功能， 以避免当您键入用户名称时， 电脑会自动完成您的密码输入。要取消「 自动完成」 功能， 请在浏览器内按一下「 工具」， 按一下「 Internet 选项」， 按一下「 内容」， 然后按一下「 自动完成」 按钮， 最后取消「 表单上的使用者名称和密码」 功能。

若需要与别人共用电脑， 请问应如何保障个人资料?

为了保障您个人户口的资料， 请紧记当您登出网上服务后必须删除浏览器内的临时档案并关闭浏览器。

保护账户资料有何秘诀？

我们建议各电子交易服务用户应该采取以下程序，以维持高度安全标准：

  • 1定期透过网页更改用户密码。
  • 2切勿向任何人透露密码。
  • 3设定一个容易记得但别人难于以猜测的密码，请勿用重覆数字、出生日期、电话号码或身份证号码来做密码。
  • 4完成网上交易后便退出互联网浏览器，有助避免其他人使用回到前一页」的功能来阅览阁下的账户资料。
  • 5经常键入网址(www.htisec.com)或使用阁下建立之位址连接来浏览大福网页

更改交易要注意些什么？

未能即时成交的增强限价盘将成为相同价格的限价盘， 更改增强限价买／卖盘时只可输入不高于卖出／不低于买入盘的最佳价格， 否则系统将拒绝有关更改．
