海通国际证券集团有限公司及其附属公司(「海通国际证券集团」) 之个人资料私隐政策

海通国际证券集团一直致力为客户提供最佳之服务。

要达到此一目的， 其中一个途径就是利用客户的资料， 为客户提供最方便的途径， 获得合适的产品和服务， 我们亦明白到客户对于其个人资料的使用非常关注。保护客户资料乃是我们一直认真处理的事项。因此， 海通国际证券集团订立了以下守则， 矢志承诺对客户的资料保密：

a. 无论任何时候， 个人名义的客户均需要向海通国际证券集团提供个人资料(「资料」)， 以作个人资料(私隐) 条例(《香港法例》第486 章) 下之用途。有关客户的资料(或其他资料) 可能用于以下用途：

  • 1向客户提供之服务及设施之日常运作；
  • 2进行信贷审查；
  • 3协助其他机构进行信贷审查；
  • 4确保客户的信用维持良好；
  • 5设计供客户使用之金融服务或相关产品；
  • 6向客户推广金融服务及相关产品；
  • 7厘定拖欠客户或客户拖欠之债务金额；
  • 8根据任何法例或规例之规定， 符合作出披露之要求；
  • 9及与任何前述部份有关之任何用途。

b. 海通国际证券集团持有之客户有关的资料(及其他资料) 将会保密， 但海通国际证券集团可向以下人士披露所有资料(及其他资料)， 而客户同意披露所有资料(及其他资料) 亦是海通国际证券集团向客户提供服务、 产品及资料之条件：

  • 1任何向海通国际证券集团提供有关其业务运作之行政、 信贷资料、 债务追讨、 电讯、 电脑、 缴款或其他服务之高级职员、 雇员、 代理、 承包商或第三者；
  • 2客户已有或拟与之进行交易之任何金融机构；
  • 3监管当局及其他有关政府机构；
  • 4任何有责任为海通国际证券集团保密之其他人士， 包括海通国际证券集团以内承诺保持此等资料机密之公司。
  • 5海通国际证券集团内公司之间对客户之资料使用， 须依据严格之内部安全标准、 保密政策及适用法律。
  • 6我们约束雇员完全遵守该等标准、 政策及法律。
  • 7除为了进行业务、 遵守适用法律、 保护免受欺诈或作出我们认为可能符合客户利益之产品及服务优惠外， 我们不会将有关客户的资料分发予其他公司。我们亦可依据适用法律向监管当局及执法人员提供资料。
  • 8我们订立了极高标准， 以保护客户的资料免受未经授权之更改或破坏。

海通国际证券集团可依据个人资料(私隐) 条例及其他适用法律进行以下活动：

a.将客户提供或有关客户之任何资料或其他资料， 与海通国际证券集团或任何其他人士持有之资料(或其他资料) 进行配对、 比较或交换，以作一下用途：

  • 1信贷审查；
  • 2资料(或其他资料)核对；
  • 3或， 提出或核对资料(或其他资料)， 该等资料可能用于在任何时候采取对客户或任何其他人士不利的行动；
  • 4将此等资料(或其他资料) 转往香港以外任何地方(不论在香港以外处理、 持有或使用此等资料(或其他资料))。

b.根据及依据个人资料(私隐) 条例之条款， 每位客户均有权：

  • 1检查海通国际证券集团是否持有有关客户之资料， 及是否有权使用此等资料；
  • 2要求海通国际证券集团更改有关客户之任何错误资料；
  • 3及确定海通国际证券集团有关资料之政策及做法， 以及获知海通国际证券集团持有有关客户之个人资料种类。

查阅及/或改正客户所提交之任何资料的要求交送交online@htisec.com

依据个人资料(私隐) 条例之条款， 海通国际证券集团有权就处理任何资料索取查阅之申请收取合理费用。

在海通国际证券集团， 有关客户之资料只会用于进行本身业务的合法用途， 以提供优质服务， 及进行产品设计及优惠， 从而显示我们明白客户及其需要。

随着我们会在新科技年代进一步发展新产品和服务， 我们会继续竭力确保客户的资料会被正确使用及受到适当保护。
