「易轉數」服務適用於所有使用透過itrade.htisec.com使用網上「繳費靈」作轉賬海智iTrade客戶。在服務時間內所作之轉賬，款項將即時存至海智iTrade戶口。

進行網上買賣而遇上金額不足時，透過itrade.htisec.com的「易轉數」服務以網上「繳費靈」戶口作即時轉賬，立即可傳送買賣指令。

利用中國銀行(香港)服務於網上轉賬，必須從海智iTrade自主網內「中國銀行(香港)智達銀行服務」登入，以取得足夠資料完成即時轉賬過程。

*按此參閱繳費詳情。

於東亞電子網絡銀行網頁之「電子繳費」付款。

「繳費靈」用戶如已設「繳費靈」八位數字網上密碼，可於服務時間內透過www.ppshk.com進行轉 賬，款項亦將於下一交易日更新至證券戶口。
