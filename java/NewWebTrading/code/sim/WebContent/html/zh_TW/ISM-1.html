智慧型選股 賬戶服務 > 智慧型選股

「智慧型選股」為客戶提供一個完善的網上股票資料庫，透過先進的濾股功能，配合個人化需求，協助客戶洞悉先機，運籌帷幄。


產品介紹

現在「智慧型選股」再新增多項功能，包括：10年盈利及派息記錄，業務回顧與展望，股份拋空及新股透視。預設的報告包括: 環球金融市場及環球股票市場、市場脈搏、異常買賣、公司備忘、交易縱覽、公司動向。

  • 110年盈利及派息記錄 - 提供更詳盡的公司歷史數據(舊版只提供5年)。
  • 2業務回顧與展望 - 由公司年報概括公司過往一年業務及來年展望。
  • 3股份拋空 - 投資者可以此數據判斷異常買賣之情況或買賣信號啟示。
  • 4新股透視 - 提供詳盡新股資料包括保薦人表現比較。另外亦記錄有關新上市公司及認股證之重要資訊。除了可以搜尋一般資料如上市時間表及上市後之股價表現，亦能獲取對新股股價敏感之重要消息如股份禁售期等。
  • 5環球金融市場及環球股票市場 – 提供環球市場價格走勢，包括各主要世界股市指數、外匯市場、美國債券市場及及主要期貨市場等等。
  • 6市場脈搏 – 透過多種衍生工具市場及現貨市場數據，洞悉市場走勢，包括「指數成份股表現」、「股票排行榜」、「股證排行榜」、「市場成交分析」及「衍生工具交易概覽」。再加上其他技術及基本數據，便可觀察市場的轉向。
  • 7異常買賣 – 股份回購、股份拋空及大手成交數據可令投資者得悉知情人士之異常買賣紀錄，以另一個角度探究個別股份的潛在價值，例如龐大累積的股份拋空可以是一個買入的信號。
  • 8公司備忘 – 提供上市公司之基本數據，投資者亦可選取一些業績由虧轉盈、虧損收窄、盈利增長的公司進行更深入的了解。
  • 9交易縱覽 – 包括所有股票之買賣安排 (如平行買賣、公可停牌、更改公可名稱等等)及交易警報(如異動股價及成交、財務預警、傳媒報道澄清等等)。
  • 10公司動向 – 一些非敏感性的數據如股東大會、更換董事等等。
