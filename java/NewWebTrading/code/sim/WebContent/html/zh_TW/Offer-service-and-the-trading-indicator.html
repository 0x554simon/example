交易類型目錄

競價盤(AO) 競價盤是沒有指定價格的買賣盤，只適用於開市前時段(詳情可參閱交易時間表)。在輸入AMS/3 系統後按最終參考平衡價格進行對盤。競價盤享有較競價限價盤優先的對盤次序及根據時間先後次序按最終參考平衡價格順序對盤。於開市前時段輸入而未完成的競價盤會於持續交易時段開始前自動取消。
競價限價盤(ALO) 競價限價盤是指在交易日上午7時至9時15分開市前時段內以指定價格作單一價格競價，上午9時15分後不接受新競價限價盤，已輸入之競價限價買賣盤不得更改或取消。買賣盤將以「參考平衡價格」進行配對。輸入價不偏離按盤價九倍以上的競價限價盤，將一概轉為限價盤，轉往當日持續交易時段進行對盤。 「參考平衡價格」(IEP) 為於對盤時段可以成交最多股數的價格，「參考平衡成交量」(IEV)則指可按「參考平衡價格」成交的股數。有關參考數位將於大利市機顯示。
增強限價盤 (ELO) 增強限價盤最多可同時與五條輪候隊伍進行配對（即最佳價格隊伍至距離4個價位的第5條輪候隊伍），只要成交的價格不差於輸入價格。增強限價盤只適用於持續交易時段(早上 9時30分 至下午4時)。早上9時15分後輸入的買賣盤都會處於”已接收”狀態，並於上午 9時30分 後存於所輸入價格的輪候隊伍中。未能於持續交易時段完成的增強限價盤會於收市後自動被取消。更改未成交沽盤的輸入價格不可低於最佳買入價5個價位（或以上），而買盤的輸入價格不可高於最佳沽出價5個價位（或以上）。任何未能即時成交的增強限價盤，將一概視為限價盤，存於所輸入價格的輪候隊伍中。
特別限價盤
(SLO) 		特別限價盤最多可同時與5條輪候隊伍進行配對。未能及時成交或部分成交的SLO會被系統取消。 SLO 只適用於持續交易時段(早上10時至下午4時)。 SLO 盤種並不支援 AON。 「特別限價盤」的下單指令只能在「證券至尊寶」及「網上證券交易」有效執行。其中流動交易，Mango Combo，3G流動電話及音頻電話暫時不能顯示部份資訊 (如“交易類型”, “全額或取消” 等)。
條件指示盤(Conditional Order) 條件指示盤適用於增強限價盤(ELO)和競價限價盤(ALO)。海通國際證券客戶下ELO單或ALO單時，通過選擇條件指示盤可自行設置觸發條件。當現價到達“觸發價”, 交易才會以下單價格送出市場。
注意 :
1.設置觸發價等於現價，系統立即將此單送入市場。
2.設置觸發價不等於下單價時，當現價漲跌過程中達到了觸發價位，系統將會以下單價將此單送入市場。
3. 買賣價格的預設值必須設定在啟動價格[對上/對下]的15/2 個價位

條件指示盤

「條件指示盤」的買賣價位會因被輸入的啟動價格而有以下限制:

當投資者希望在某股票升穿或跌穿某一價位時，立即以一指定限價盤入市買賣開倉或平倉，準確地捕捉市場機會，可選用海通國際證券證券至尊寶或證券網上交易的條件指示盤功能：當該股票的現價(Nominal Price)到達投資者自定的觸發價位時，系統便自動將投資者指定的限價盤送出市場待成交。條件指示盤分為「條件指示盤-升穿」及「條件指示盤-跌穿」。

「條件指示盤」的有效期是多久?

所有條件指示盤只在設定當日有效，如該買賣盤在當日收市前未能成交，有關交易指示將被取消。 「條件指示盤」可於開市前時段(09:30 – 09:50)根據參考平衡價格被觸發並執行。另外, 條件指示盤也適用於「下一交易日」時段, 收市後發送的條件指示盤會被儲存至下一交易日早上執行, 當日有效。有關「下一交易日」盤的詳情, 請參閱“有效至「下一交易日」盤種的定義及操作模式”。

「條件指示盤」有什麼限制嗎?

「條件指示盤」的買賣價位會因被輸入的啟動(觸發)價格而有以下限制:

  啟動(觸發)價格 買賣價格與啟動(觸發)價格的關係
啟動買入-下限 沒有限制 啟動價格對下15個價位
啟動買入-上限 沒有限制 啟動價格對下2個價位
啟動買入-上限 沒有限制 啟動價格對下15個價位
啟動買入-下限 沒有限制 啟動價格對下2個價位

買賣價格「預設值」設定在啟動(觸發)價格「對上/對下的15/2個價位」而不是根據現時市場掛牌的24/4個價位的原因在於減少當「條件指示盤」被觸發送出市場時因股票現價(Nominal Price)變動而被拒絕的情況發生。

我可以在什麼平台使用「條件指示盤」下單指令？

「條件指示盤」的下單指令只適用於「證券至尊寶」 及「證券網上交易」。閣下將無法透過其他網上交易管道(例如：流動交易，3G流動電話，音頻電話及Mango Combo) 提交「條件指示盤」下單指令，但仍可修改或取消「條件指示盤」的股價和數量(Mango Combo 除外)。

「條件指示盤」與「止蝕盤」/「止賺盤」一樣嗎?

「條件指示盤」與「止蝕盤」/「止賺盤」是不相同的。 「條件指示盤」是當該股票現價(Nominal Price)到達客戶設定的啟動價位時，系統會自動將客戶指定的限價盤送出市場待成交，因股票市場並非如期貨市場般客戶集中在指定產品交易及沒有價位元元限制，故此送出市場的限價盤可能會因為市場流通性不足而未能成交，因此「條件指示盤」即使在觸發後送出市場亦有機會因買賣盤疏落未能成交而達不到止賺/止蝕效果。

指令會被拒絕或因其他原因未能執行嗎?

「條件指示盤」與「止蝕盤」/ 視乎市場情況而定, 即使有關股價已達至指定的觸發水準, 指令可能因股價變動, 系統故障及其它不可預計之情況而未能執行。即使「條件指示盤」已被觸發，該盤亦可能因價格波動太大而被交易所拒絕。

我可以更改「條件指示盤」指令嗎?

「條件指示盤」與「止蝕盤」/ 系統於交易日早上便會檢查「條件指示盤」的觸發條件，「條件指示盤」一經觸發，啟動價格便不能修改( 早上7 時至9時，已觸發的「條件指示盤」仍會是「已接收」的狀態，上午9時至下午4時，已觸發的「條件指示盤」會變成「挂盤中」的狀態)。
